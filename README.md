# AirVis ———— 2025年全国空气质量数据可视化分析系统

AirVis 是一个多层次、多维度的全国空气质量数据可视化系统。项目基于 2025 年全国主要城市的真实监测数据，利用 D3.js 引擎实现了热力图、星云图、山峦图、雨云图等交互组件，支持从全国概览到城市细节的渐进式探索。

### 数据来源
空气质量数据：来源于 QuotSoft Air 历史监测数据及中国气象数据网。
地理空间数据：基于 China Map 开源项目及中国国家地理网行政区划资料。

### 运行教程
确保已安装 Node.js，在项目目录下执行：
1. `npm install`
2. `npm start`
